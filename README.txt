Projet C++ de Thomas HAUTIER et Richard BUNEL.

Pour compiler le projet, se rendre dans "Source", puis exécuter la commande "make" ou "make all".

Ceci crée un exécutable nommé "Projet".

La commande "make clean" le supprimera.

Pour lancer le projet, exécuter "./Projet".

Il sera demandé quel jeu lancer ainsi que les dimensions.

Pour jouer, les entrées "h","d","b" et "g" correspondent respectivement aux déplacements vers le haut, la droite, le bas et la gauche.

L'entrée "i" fait jouer le joueur "stupide", aléatoirement dans l'une des quatre directions précédentes.
